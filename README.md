# UI/UX Experiment 6

## Aim
To understand and apply UX design processes including solution ideation, user story creation, scenario building, task flow mapping, and information architecture design.

---

# 1. Solution Ideation

## Problem
Many users find it difficult and time-consuming to book doctor appointments using hospital mobile applications.

## Crazy 8s Ideation

The Crazy 8s technique was used to generate multiple quick ideas for solving the appointment booking problem.

### Ideas
1. One-tap quick appointment booking
2. Doctor availability calendar
3. AI symptom checker recommending doctors
4. Voice based appointment booking
5. Emergency quick consultation button
6. Filter doctors by specialization
7. Appointment reminder notifications
8. Live waiting time tracker

# 2. User Stories

User stories help understand the user's needs and goals.

### User Stories

1. As a **patient**, I want to **find available doctors quickly**, so that **I can book appointments without wasting time**.

2. As a **working professional**, I want to **view doctor availability schedules**, so that **I can choose a convenient appointment time**.

3. As a **new user**, I want to **search doctors by specialization**, so that **I can easily find the right doctor**.

4. As a **patient**, I want to **receive appointment reminders**, so that **I do not miss my consultation**.

5. As a **patient**, I want to **book appointments online**, so that **I can avoid long hospital waiting queues**.

---

# 3. User Scenario

### Selected User Story
As a patient, I want to find available doctors quickly so that I can book appointments easily.

### Scenario

Priya is a 28-year-old software engineer who suddenly feels unwell before leaving for work. She opens the hospital mobile application on her smartphone while sitting at home. She feels worried because she needs medical consultation quickly but does not want to spend hours waiting at the hospital.

Using the search feature in the app, she filters doctors by specialization and views available appointment slots. She selects a doctor with a convenient time slot and confirms the booking within seconds.

After receiving the appointment confirmation notification, Priya feels relieved because she was able to secure medical help quickly without waiting in long hospital queues.
